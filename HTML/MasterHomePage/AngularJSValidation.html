Angular JS Validation

In the previous article, we talked about Angular JS and its use in forms. However, this might raise some questions, especially among beginners to the question: How to validate the form when this is done? In today's article, you'll get to know about AngularJS form validation. Keep reading to know more about it.

Again, you need to initialize the Angular JS script. We have provided one for you, check our previous article to know how to set it up in your page.

Let's get started!

Before your form can be validated, it is important to put the tag <form (parameters here)></form> in your page, so that the page can know the form element to call AngularJS script.
Now, once you have the form, you don't need to go anywhere else. Just remember some parameters that is used in forms to get AngularJS up and working for your own form:
  • ng-submit: used in form tag, informing what to do when you submit the form.
  • ng-model: used majorly in inputs, telling the script to set the name in the parameter to be used for other events that user specified.
  • ng-show: message to show when conditions in it are met.
  • ng-pattern: specifying the pattern. This must be preconfigured in the JavaScript file linked to this HTML file.
  • ng-minlength: specifying the minimum accepted length of the input. Inputs that have length less than this value will not be accepted.
  • ng-maxlength: specifying the maximum accepted length of the input. Inputs that have length more than this value will not be accepted.
Besides, there are some preconfigured parameters that you need to know:
  • $touched: indicating that the input is touched.
  • $untouched: indicating that the input is untouched yet.
  • $pristine: indicating that the input hasn't changed since the page is loaded.
  • $dirty: indicating that the input has changed since the page is loaded.
  • $valid: indicating that the input is valid.
  • $invalid: indicating that the input is invalid.
You can use them in your form to make it be automatically validated every time you input in the form (or best called, it is validated immediately). Check the following form for example:
