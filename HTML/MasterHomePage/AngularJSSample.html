Angular JS Introduction

Angular JS is one of the most popular kinds of JavaScript used by many IT professors and programmers to make the most of the web. More importantly, it is open-source and free, so Angular JS is more and more popular across the web nowadays. If you want to do it yourself, this is the right place for you. Let me guide you in today's article!

Firstly, you need to initialize the Angular JS script. Here we have locally stored one for this script, but if you want to use it directly, you might want to read the original source code here.
Now that you have the source code, check the header - which is the html tag. The ng-app is provided in that tag, with the name corresponding to the angular.module name in the JS file named AngularJSPage.

Let's control it!

After you have initialized the module (watched the source code yet? It was myApp), we have to add controls key to it. Right below this div tag in source code, we have another ng-controller parameters named "firstController", which is the same as the name of the first parameter in app.controller. The second one matches myController, which is a function called to the div to execute the angular control blinder. Watch the example below:

Your name: {{name}}
Your class name: {{className}}
Concat your name and class name: {{name + " " + className}}


Well, the above example uses brackets to encase your desired text. There is another example, which makes the use of the typed text above, that is p tag with ng-bind parameter. Type your name and class in the input boxes above and check the result below.
Name:

Class:

Check the code! The function mentioning in above example uses $scope.name and $scope.className. When you first load into this page, these inputs are left blank, and since it is required, the borders are red. When you type any characters in those boxes, the input border turns green.
Another example is that you can make your own array and embed them directly into your page. Here's an example: When specifying std1.name is "Le Ngoc An" and std1.className is "T2301E", it returns like this:
Your name: {{std1.name}}
Your class: {{std1.className}}
Here's a more simple example. Just type 5+6 in brackets and it returns: {{5+6}}.
If you want to init a test directly, it's OK! Watch this example when we put test=1 (which is a number) and test1='T2301E' (which is a string):
test: {{test}}
test1: {{test1}}

More complexity incoming...

Now, let's turn into more complex one. Check the example below and press "Click here to plus one".

The current number is {{count}} (default 0).

Check it out! When you click on the button, it increases the value of $scope.count. The default value of that parameter is 0, and to make this works, we have another parameter namely, ng-click. And when we click on it, the button triggers the command to plus one on the count value, that's how it works.
Got it? Let's go to another example, which is quite interesting to find out.

Is making a list easy?

Well, just a problem of an array. Check the example - well, not an example, but a bulleted list below:
  • {{std}}
Well, that's how we make a list! We just need ng-repeat. But not like simple example above, we need an array, with each encased in 'this', and all [encased here], seperated with comma. Once you made it, the list flows as a breeze!

Let's make a functional table!

But not just a normal table... here's a good table to learn about!
STT Name Price Created at
{{$index + 1}} {{product.name | uppercase}} {{product.price | currency:"VND":4}} {{product.createdAt | date:"yyyy-MM-dd"}}
Showing from {{offset+1}} to {{offset+limit}} out of {{length}}.

Such a complex list, right? Again, by default, the table limit is 5, as specified by $scope.limit, and the table offset is 0, as specified by $scope.offset. But that's not enough. Both the offset and limit is changeable thanks to the back and next button, which minus or plus the current value in limit to the table using ng-click parameter; as well as the "Limit the list to" box, which directly limits the entries that appears on the board using ng-limit parameter. The table is then configured by using "limitTo:limit:offset" option in ng-repeat parameter. The default soft is by price, as specified in the "orderBy:'price'" option. Check the code of this article to see whether it was right or not.
